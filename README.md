# Equipe32 & Tessan 

> **Hackathon Tessan x Snowflake : Transformer les cabines médicales en centres de diagnostic respiratoire intelligents grâce à l'IA.**

---

## Le Défi
En France, où l'accès aux soins devient un enjeu critique, les 1 600+ cabines médicales **Tessan** permettent de rapprocher la médecine des patients. 

L'objectif de ce projet est de doter le stéthoscope connecté de ces cabines d'un "cerveau" capable de déceler et classifier 5 états respiratoires en temps réel : **Sain, Asthme, BPCO, Bronchite, et Pneumonie** à partir de sons (fichiers WAV).
